# **`Moudule`의 종류**

## **1. Built-in Core Module**

내장되어 있는 코어 모듈.

대표적으로 `Node.js module`이 있다.

```
const path = require('path');
```

## **2. Community-based Module**

대표적으로 `NPM`이 다루는 패키지 매니저를 통해 다루는 모듈들이 있다.

`jQuery` 등이 2번에 해당.

---

## **3. Local Module**

특정 프로젝트에서 정의된 모듈.

---

> 즉, 모듈들의 사용 방식과 위치에 따라 다양하게 구분할 수 있다.
