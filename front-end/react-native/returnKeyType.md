## ReturnKeyType

`TextInput`에는 `returnKeyType`이라는 것을 지정할 수 있다.

이는 키보드로 입력 후 엔터의 타입을 지정해주는데, 플랫폼별로 공통되는 것도 있고, 다른 것도 있으니 주의해야 한다.

### 공통

+ `done`: 완료
+ `go`: 이동
+ `next`: 다음
+ `search`: 검색
+ `send`: 보내기

### ios

+ `default`: 기본
+ `emergency-call`: 긴급전화
+ `google`: 검색
+ `join`: 가입
+ `route`: 이동
+ `yahoo`: 검색

### android

+ `none`: 일반 엔터
+ `previous`: 뒤로
