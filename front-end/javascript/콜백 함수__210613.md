# **콜백 함수**

함수의 변하지 않는 공통 로직은 미리 정의하고, 경우에 따라 변경되는 로직을 추상화하여, 매개변수를 통해 함수의 내부로 전달되는 함수

> **cf) 고차함수:**  
> 매개변수를 통해 함수의 외부에서 콜백 함수를 전달받은 함수

```
function repeat(n, f) {
    for (var i = 0; i < n; i++) {
        f(i)
    }
}

var logAll = function(i) {
    console.log(i);
}

// 반복 호출할 함수를 인수로 전달
repeat(5, logAll); // 0 1 2 3 4

var logOdds = function (i) {
    if (i % 2) console.log(i);
};

repeat(5, logOdds);
```

- 콜백함수는 고차 함수에 의해 호출됨.
- 고차함수는 매개변수를 통해 전달받은 콜백 함수의 호출 시점을 결정해서 호출.

  > **따라서 콜백함수는 `()`을 달지 않는다. 단지 고차함수가 콜백함수의 호출 시점을 결정한다.**

- 장점: 중첩함수와 달리 외부에서 내부로 전달되므로 자유롭게 교체할 수 있다는 장점

만약 고차 함수에만 호출되는 콜백 함수라면, 다음과 같이 익명 함수 리터럴로 생성하는 것이 일반적이다.

```
repeat(5, function(i) {
    if(i % 2) console.log(i);
}); // 1 3
```
